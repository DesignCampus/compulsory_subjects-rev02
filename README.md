# DESIGN CAMPUS 教材

デザキャンのコーディングのライブ授業前期の教材です。  

## ファイル構成

```
├ [doc] : 講義についてのドキュメント
│   ├ [img]：ドキュメントで使用する画像を格納
│   ├ doc-Xw.md：各週の講義のスライド
│   ├ my-theme.css：スライド用CSS
│   └ XXX.pdf：各週のスライドをPOF化したもの
│
├ [example] : 講義で使ったサンプル
│   ├ [img]：サンプルで使用する画像を格納
│
├ .gitignore : 管理用のファイル
└ README.md : このファイル
```
