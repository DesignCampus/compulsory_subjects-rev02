## JavaScript

--

ライブ授業ではjQueryは使いません

--

Vanilla.jsでいこう

--

- jQueryを使わないと面倒だったことが、Vanillaで普通にできるようになってきた
- 現在は様々なフレームワークやライブラリが使われるので、Vanillaが分かっているほうが応用が利く

--

### イベントで処理を走らせる

--

#### jQuery
```JavaScript
$('セレクタ等').on('イベント', function() {
    // 処理
});
```

#### Vanilla
```JavaScript
'セレクタ等'.addEventListener("イベント", function() {
    // 処理
});
```
